使用redis Stream来做一个简单的队列

Published Jan 6, 2023 by in at https://www.cleanbug.cn/redis/stream/

Redis 5 新特性引入了新的数据结构Streams 。它使 5.x 版本迭代中，Redis 作为消息队列使用时，得到更完善，更强大的原生支持，其中尤为明显的是持久化消息队列。

同时，stream 借鉴了 kafka 的消费组模型概念和设计，使消费消息处理上更加高效快速。本文就 Streams 数据结构中常用 API 进行分析。

java开发，目前在一家区块链公司工作，这里分享一些日常使用到的技术
